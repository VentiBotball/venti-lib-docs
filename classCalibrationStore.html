Detailed Description

template<typename T>
class CalibrationStore< T >

A template class that stores and retrieves calibration values for sensors.

This class is used to manage the calibration values for sensors. Each calibration value is associated with an id, and the class stores these values in a map. The calibration values are initiated by calling the corresponding initiator function.

Template Parameters
TThe type of the calibration value. It can be any type that represents the calibration data for the sensors, such as short, float, or bool.

CalibrationStore< T >::CalibrationStore ( unordered_map< string, function< T()> > initiator_funcs)

Constructs a CalibrationStore object with a set of initiator functions.

Initializes the CalibrationStore with a set of initiator functions that generate calibration values. The initiator functions are mapped to unique id strings. These functions are expected to return values of type T (e.g., short, float).

Parameters
initiator_funcsA map where each key is a calibration id (string), and the corresponding value is a function that returns a calibration value of type T.

Detailed Description

Provides advanced motor control using threaded PID loops and optional easing functions.

The ImprovedMotor class wraps a physical motor with support for:

  • PID-controlled velocity and position movement
  • Threaded background motion execution
  • Easing functions for smoother acceleration
  • Integration with a SensorRegistry for position tracking

All motion is non-blocking and controlled via a background thread. PID can be configured dynamically.

Example: waiting for motion to complete using is_motor_done()

ImprovedMotor motor(...);
motor.move_pid_distance(0.5, 1000);
while (!motor.is_motor_done()) {
std::this_thread::sleep_for(std::chrono::milliseconds(10));
}
motor.freeze(); // Optionally stop after arrival

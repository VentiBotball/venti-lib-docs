nlohmann::json fetch_data ( const std::string endpoint)

Fetches JSON data from a given API endpoint.

This function sends an HTTP GET request to the specified URL using libcurl, retrieves the response body, and parses it into a JSON object using nlohmann::json.

Parameters
endpointThe API URL to fetch data from (e.g., "https://api.example.com/data").
Returns
A nlohmann::json object containing the response data. If the request fails or the response is not valid JSON, an empty JSON object is returned.
Note
Requires libcurl and nlohmann::json to be properly linked in your build system.
See also
https://github.com/nlohmann/json
https://curl.se/libcurl/c/
// Example usage:
#include "fetch_data.hpp"
#include <iostream>
int main() {
std::string url = "https://api.agify.io/?name=michael";
nlohmann::json data = fetch_data(url);
if (!data.empty()) {
std::cout << "Estimated age: " << data["age"] << std::endl;
} else {
std::cerr << "Failed to fetch or parse JSON data." << std::endl;
}
return 0;
}

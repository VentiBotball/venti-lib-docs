Detailed Description

Defines the abstract motor controller interface and concrete implementations for omni and differential robots.

This file provides a unified interface (MotorController) for robot movement logic, along with two implementations:

  • MecanumMotorController: controls a 4-wheel omni-directional drive with Mecanum wheels.
  • DefaultMotorController: controls a 2-wheel differential drive robot.

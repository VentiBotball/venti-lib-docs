void until_light ( int port,
int max_waiting = 0 )

Waits until a bright light turns on to start the robot.

This function performs an interactive two-step calibration with the user:

  1. Calibrate the light ON value by placing the light on the sensor and pressing the side button.
  2. Calibrate the light OFF value by covering the sensor and pressing the side button again.

It then waits until the sensor detects a significant increase in brightness (interpreted as the light turning on) to proceed.

If the calibration difference is too small (i.e. not enough contrast), the process restarts.

While waiting:

  • Pressing the side button restarts calibration.
  • If max_waiting > 0, the function will timeout after max_waiting milliseconds.
Parameters
portAnalog sensor port to use (e.g., 0 for analog(0)).
max_waitingOptional failsafe timeout in milliseconds (default = 0 = no timeout).

Example:

#include "until_light.hpp"
int main() {
// Wait up to 10 seconds (10000 ms) for the light
until_light(0, 10000);
// Robot starts here after light is detected
return 0;
}
